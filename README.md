# Features : 
+ Shop Characters
+ Friend (Google Play Game)
+ Chat
+ Invite
+ Matchmaking
+ Room
+ Rank
+ Ads (Unity Ads)
+ Puzzle answer question
+ Multiplayer Online (2 players in a match)

# Technologies :
+ Unity, GooglePlayGame Services, Firerbase, Photon Unity Networking 2, Unity Ads
